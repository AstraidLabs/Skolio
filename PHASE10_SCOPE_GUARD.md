# Skolio Phase 10 Scope Guard

This phase keeps the existing service-based architecture and only completes PlatformAdministrator governance capabilities.

## Explicitly Out of Scope

The following areas are not implemented and must not be reintroduced in Phase 10:

- tests workflows
- quizzes workflows
- assessment workflows
- exams workflows
- online testing workflows
- question bank workflows
- automated grading workflows
- scoring workflows
- evaluation workflow engines
- exam workflow engines
- assessment engines
- university model
- credits model
- semesters model
- subject enrollment model

## School Type Guard

Supported school types remain unchanged:

- Kindergarten
- ElementarySchool
- SecondarySchool
